# Online Quiz Management System

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript
* XAMPP Server

## Modules

* Admin Login
* Add Quiz
* View Quiz
* Add Questions
* Student Registration
* Attend Quiz
* Result Generation

## How to Run

1. Install XAMPP
2. Start Apache & MySQL
3. Import database from `database.sql`
4. Run: http://localhost/quiz_management_system
